CSS snap effect tutorial

First, you need sections for each page, and wrap them with a container

Second, inside the container class assgin the following css properties:

scroll-snap-type: y mandatory //how strictly snap points are enforced on the scroll container in case there is one. Mandatory makes the effect more obvious, it may have some problem with some mouses scroll wheel setting. Proximity may be a better option in this case.

overflow-y: scroll; // when an element's content is too big to fit in, it should scroll

height: 100vh; // match the children height
